# Prediksi-Kelulusan-Mahasiswa-Menggunakan-Support-Vector-Machine-SVM-
Proyek ini bertujuan untuk melakukan analisis data kelulusan mahasiswa menggunakan algoritma Support Vector Machine (SVM). SVM digunakan sebagai model klasifikasi terawasi (Supervised Classification) untuk memprediksi apakah seorang mahasiswa akan lulus Tepat Waktu atau Terlambat.
